## Project requirements

Python version >= 3.8

Pip version >= 23.2

## Install dependencies 

```bash
python3 -m pip install -r requirements.txt

python3 -m playwright install

```

If you have problems, install packages manually

```bash
python3 -m pip install {package-name}

```
